# Descripción de los datos

El dataset utilizado en este proyecto es de tipo simulado y representa el comportamiento del sistema de transporte masivo.

## Variables

- distancia: distancia entre estaciones
- trafico: nivel de congestión (bajo, medio, alto)
- hora: momento del día (pico, normal)
- tiempo: tiempo estimado de viaje (variable objetivo)

El dataset se construyó con el fin de entrenar un modelo de aprendizaje supervisado.
